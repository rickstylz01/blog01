The fizzbuzz for loop

So for this project all I did was create a for loop that logs out numbers 1-100. The only difference is that I also used the fizzbuzz "technique" which will alert "fizz", "buzz, or "fizzbuzz" when certain conditions are met.

In this case everytime a number is generated, it is also calculated to see if it is divisible by a 3 or 5 or a 5 and 5, in which case it would alert a specific string.

This is an example of using an if/else statements witin for loops.
