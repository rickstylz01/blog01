To solve a fizzbuzz

Question:Generate a random number between 0 and 5. If the result is divisible by 3 alert "fizz". If the number is divisible by 5, alert "buzz", otherwise print the number to the console.

The first step is to create a varaible to hold the answer value:

  • var randomNumber = Math.round( Math.random() * 5);
This variable rounds a random number and times it by 5. (in order to use a whole number).

Now we can use our "if, else" statements to determine when our alert would pop up so:

  • if(randomNumber%3 === 0) { alert("fizz"); }
This statement is saying, if the randomNumber variable generates a number that is divisible by 3, then alert 'fizz'."

The way the statement is written out is leterally saying, if the remainder of a random number, divided by 3, equals 0, then alert "fizz".

Then it's time for the "buzz" so:

  • else if(randomNumber%5 === 0) { alert("buzz"); } else { console.log(randomNumber); }
Now, according to the directions, we have to generate a random number between 0 and 15. We have the code set up so that if whatever random number generated is divisible by 3 it will alert "fizz" and if it's divisible by 5 then "buzz" will be the alert. Also our randomNumber variable is set to generate a random number, round it, and multiply it by 5 and we have to change that to 15.

The directions also state, "if a number is divisible by 3 AND 5, alert fizzbuzz and if it is 0 then alert nothing".

So in order to accomplish that we use the "&&" symbols to state that, if a random number is divisible by 3 and 5 then something should happen. So:
  • if(randomNumber%3 === 0 && randomNumber != 0) { if(randomNumber%5 === 0) { alert("fizzbuzz"); } else { alert("fizz"); } }
Literally, what this code is saying is, if the randomly generated number is divisible by 3 AND does not equal 0, Then it will check to see if the random number is divisible by 5 and if so then it will alert "fizzbuzz".

Within this bit of code from our if/else statements, You'll notice we used an if statement within another if statement, which helped in specifying the conditions we needed to meet to create the alert "fizzbuzz". Now all the conditions within this statement must be met to create the alert or else only the "fizz" alert will pop up.
