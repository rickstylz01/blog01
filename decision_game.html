I started by making an alert to set up the premise. Then I used a prompt command inside a variable, to give the player a choice. Once the player made a choice, I made so that whatever choice was made I concatenated a string to what they wrote.

For example:

  • var choice = prompt("You have two pairs of shoes, Nike, Adidas. Which do you wear?");
  • Alert("You chose " + choice

Then I created a judges scoring system using Math.round and Math.random. By puttin Math.random() inside of Math.round() it cancels out math.random() using decimals. I put this formula into a varible called score. So whenever I wanted a random number to be used as a score I simply input the variable name (score). But, I also made a score for the competition using the variable enemyScore.
